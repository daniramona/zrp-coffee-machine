Automatically expire posts at a given date

== Description ==

Easily configure posts to expire at a certain date and time. After posts expire, they can become drafts, private, deleted, or in the trash. Works without Cron enabled.

== Installation ==

1. Click "Add New" in the plugins dashboard
2. Search for "Auto Expire Posts"
3. Click "Install Now"
4. Activate the plugin

== Frequently Asked Questions ==

= What time zone is the time input in? =

It uses the browser's local time zone.

= Can a message be displayed for expired posts? =

Not at the moment. In the future, we may add a feature to remove post content without removing the post itself.
